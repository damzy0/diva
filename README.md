## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [MongoDB](https://www.mongodb.com/) or [MySQL](https://www.mysql.com/) for backend (depending on your tech stack)

### Instalation

1. **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/diva.git
    cd diva
    ```

2. **Install dependencies for frontend and backend**
    ```bash
    cd frontend
    npm install
    # or yarn install

    cd ../backend
    npm install
    ```

3. **Setup environment variables**

   - Create `.env` file in `backend/` for API keys, database URIs, and email settings.

4. **Run the development server**
    ```bash
    # backend
    cd backend
    npm run dev

    # frontend (in separate terminal)
    cd ../frontend
    npm start
    ```

5. **Visit `http://localhost:3000` to see Diva in action!**
